Mi Primer Proyecto Web 🚀
Este fue uno de mis primeros proyectos donde puse en práctica todo lo aprendido sobre las bases de la web. Es una propuesta personal enfocada en la visualización y el audio, construida totalmente desde cero.

🛠️ Tecnologías utilizadas:
HTML: Para toda la estructura del sitio.

CSS: Para darle el estilo y la apariencia visual.

JavaScript: Para toda la lógica del audio y la interactividad.

📝 Sobre el proyecto:
Este trabajo fue un reto personal para demostrar que se pueden crear experiencias multimedia de calidad usando solamente código nativo, sin necesidad de librerías externas. Me enfoqué en que la visualización y el sonido funcionaran en armonía para el usuario.
